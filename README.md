# Dash — Instant browser navigation with a single command.

Open websites, workflows, and tools in seconds using a keyboard-first launcher.

</div>

---

## What is Dash?

Dash is a keyboard-first browser launcher. Open websites, tools, and workflows instantly using simple commands and aliases.

---

## Why Dash?

Too much browser navigation is repetitive.

Dash removes the friction:
open the launcher, type a command, press Enter.

No mouse. No tab hunting. No wasted time.

---


## Quick Start

**1. Open Dash**

```
Cmd + Shift + Space
```

**2. Add your first alias**

```bash
/add gh https://github.com GitHub
```

**3. Use it**

```bash
gh  →  opens GitHub instantly
```

That's it.

---

## Commands

| Command | Syntax | Description |
|---|---|---|
| Add alias | `/add <alias> <url> [name]` | Create a new website shortcut |
| Delete alias | `/delete <alias>` | Remove an existing alias |
| List aliases | `/list` | View and manage all aliases |
| Change theme | `/theme <theme-name>` | Switch between available themes |
| Help | `/help` | Show available commands and shortcuts |
| Info | `/info` | View project and developer information |

### Examples

```bash
/add gh      https://github.com          GitHub
```

## Customizing the Shortcut

The default shortcut is `Cmd + Shift + Space`, but you can change it anytime:

1. Go to `chrome://extensions/shortcuts`
2. Find **Dash**
3. Set your preferred keybinding
